🚨 **Problem Statement**


Urban trees often fail due to pollution, heat stress, and poor soil conditions.  
There is no intelligent system to predict survival beforehand.  
Cities lack data-driven tools for sustainable plantation management.


---

💡 **Solution**


GreenPulse AI is an AI-powered dashboard that:
Predicts tree survival probability (%)
Analyzes environmental conditions
Provides explainable insights
Suggests smart sustainability actions


---

🧠 **Machine Learning Model**


Random Forest Regressor
Regression model predicting survival %
Trained on environmental + tree health dataset

---

📊 **Features**


Environmental Input Panel
Survival Probability Gauge
Risk Indicator
AI Explanation
Smart Recommendations
City & Tree Presets
Mobile Friendly UI
